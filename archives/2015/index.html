Mac OS 下搭建Hexo博客环境

搭建步骤如下

  1. 注册GitHub账户
  2. 创建仓库
  3. 安装nodejs
  4. 安装git
  5. 安装Hexo
  6. 相关配置

1.注册GitHub账户

  1. 如果你已经拥有GitHub账户，请跳过此步骤。
  2. 打开 https://github.com 点击 [Sign up fo GitHub] 按钮。注册成功后请牢记您的用户名和密码
  3. 登录GitHub。

2.创建仓库

  1. 登录GitHub之后点击 [ + New repository ] 按钮新建一个仓库，用来存储博客内容，这里Repository name 填写 [你的登录名].github.io 不能用别的名字。比如，你的登录名是 cy1992s 那这个仓库的名字必须是 cy1992s.github.io。如下图
  2. 填写完信息之后，点击[Create repository]按钮，完成创建

3.安装nodejs

  1. 因为Hexo是基于nodejs的，所以必须安装nodejs。
  2. 安装方式有好多种，这里不再一一赘述.
  3. 打开连接 http://nodejs.org/download/ 我们会看到 Mac OS X Installer(.pkg) 这一项，点击后面的连接 Universal下载安装文件。如下图
  4. 下载完成之后点击直接安装即可。
  5. 安装完成后，打开终端，输入下面的命令
    $ node -v
    如果nodejs安装成功的化，你会看到命令行显示出nodejs的版本

4.安装git

  1. 安装方式有好多，这里也不再一一赘述
  2. 如果你已经安装了XCode，就不需要安装git了，如果没有安装Xcode,需要下载并安装
  3. 打开 http://git-scm.com/download/mac 下载git,并安装。

5.安装Hexo

  1. 在电脑中任意位置创建一个文件夹,用来存放博客文件。如：/Users/tofu/Desktop/Blog
  2. 打开终端，输入npm install -g hexo 如果有权限问题，请尝试sudo npm install -g hexo。npm是nodejs的包管理工具
  3. npm会下载hexo。
  4. 初始化hexo 进入步骤1中创建的目录，如：cd /Users/tofu/Desktop/Blog
  5. 输入hexo init
  6. 输入npm install
  7. 输入hexo generate
  8. 输入hexo server 会看到提示信息 Hexo is running at http://localhost:4000/. Press Ctrl+C to stop. 这表示已经博客已经搭建好了。我们在浏览器中打开 http://localhost:4000/ 查看了

6.相关配置

  1. 以上所有步骤结束之后，我们打开/Users/tofu/Desktop/Blog目录，会发现多了好多文件，打开_config.yml,更改相关的配置信息。参考：http://zipperary.com/2013/05/29/hexo-guide-3/

  2. 在文件的最后找到，deploy配置项，修改为

repository：是你在github上创建的仓库的git地址。请根据实际情况更改。
保存文件
